## ❓ Instruções Básicas

O aplicativo permite simular a execução de programas no modelo de CPU NEANDER.

* Insira instruções no editor
* Execute passo a passo ou continuamente
* Visualize registradores e memória em tempo real

## 🔒 Política de Privacidade

Este aplicativo não coleta, armazena ou compartilha dados pessoais dos usuários.

## ⚠️ Problemas comuns

* Caso o app não execute corretamente, reinicie o dispositivo
* Atualize para a versão mais recente disponível
